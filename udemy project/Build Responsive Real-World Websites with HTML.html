All modern websites and web applications are built using three fundamental basic languages: HTML, CSS, and JavaScript. these are the languages of the web.

what is html?

html stands for HyperTextMark Language. html is the standard markup language for creating web pages and web applications. with html you can create your own website.

HTML consists of elements that describe different types of content: paragraphs, links, headings, image, video, etc. Web browers understand HTML and remder HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. the opening tag
  2. the closing tag
  3. the actual element
